## 🎨 项目简介

此项目是给我**亲爱的女朋友**做的**拼豆图纸生成工具**是一款纯前端的在线工具(因此**不会上传用户图片或者隐私**，放心使用)，专为拼豆爱好者设计。用户只需上传一张图片，即可快速生成带有网格线、色号、行列编号的拼豆图纸，支持多种尺寸调整和颜色数量控制，完美适用于制作拼豆艺术作品。

## ✨ 主要功能

### 📷 图片处理
- **图片上传**：支持常见的图片格式（JPG、PNG等）
- **尺寸调整**：支持多种预设尺寸（30×30、50×50、80×80、100×100、150×150）
- **实时预览**：即时查看转换效果

### 🎨 颜色控制
- **专业颜色匹配**：使用CIEDE2000颜色差异算法，精确匹配拼豆颜色
- **颜色编号系统**：自动为每种颜色分配字母编号（A、B、C...AA、AB...）
- **颜色量化**：减少颜色数量同时保持视觉效果

### 📐 图纸定制
- **网格线显示**：清晰的网格线帮助定位
- **行列编号**：显示行号和列号便于参考
- **色号标注**：每个格子显示对应的颜色编号
- **字体自适应**：根据格子大小自动调整字体大小

### ✏️ 自定义编辑功能
- **编辑模式**：进入自定义格子模式
- **单格编辑**：点击任意格子修改颜色
- **颜色选择器**：提供彩虹色盘和渐变色选择
- **撤销重做**：支持撤销和重做操作
- **重置功能**：一键重置所有修改
- **保存功能**：保存当前修改的颜色

### 🖼️ 显示选项
- **色号开关**：可选择是否显示色号
- **行列号开关**：可选择是否显示行列号
- **网格线开关**：可选择是否显示网格线

### ⚡ 性能优化
- **纯前端处理**：无需服务器，所有处理都在浏览器完成
- **防抖机制**：避免频繁处理导致卡顿
- **智能缩放**：使用双线性插值保持细节
- **响应式设计**：适配各种屏幕尺寸
- **拖拽缩放**：支持画布拖拽和缩放查看

## 🚀 快速开始

### 环境要求
- Node.js >= 14.0.0
- npm 或 yarn

### 安装步骤
```bash
# 克隆项目
git clone https://github.com/MiManchi03/perler-pattern-generator.git
cd perler-pattern-generator

# 安装依赖
npm 安装

# 启动开发服务器
npm 运行开发

# 构建生产版本
npm 运行构建
```

### 使用说明
1. **上传图片**：点击上传按钮选择图片
2. **调整参数**：设置目标尺寸和显示选项
3. **预览图纸**：查看生成的拼豆图纸
4. **自定义编辑**：点击"自定义格子"进入编辑模式，点击格子修改颜色
5. **下载图纸**：点击"下载高清图纸"保存为PNG格式
